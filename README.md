# Masterclass: Tax and Transfer Policy Evaluation

This folder contains a compact 1.5-day masterclass:

- `lectures/session_1/`: GETTSIM structure, `main()` workflow, diagnostics, and personas.
- `lectures/session_2/`: policy reforms via parameter and function changes, plus exercises.

Useful links:

- GETTSIM documentation: https://gettsim.readthedocs.io/en/stable/
- GETTSIM how-to on policy environment modifications: https://gettsim.readthedocs.io/en/stable/how_to_guides/modifications_of_policy_environments.html
- GETTSIM GitHub: https://github.com/ttsim-dev/gettsim
